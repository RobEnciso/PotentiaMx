## 📐 ESPECIFICACIONES TÉCNICAS

### Tamaños Requeridos:

#### **Digital:**
- Favicon: 32x32px, 64x64px
- App Icon (iOS): 180x180px, 1024x1024px
- App Icon (Android): 512x512px
- Open Graph: 1200x630px
- Twitter Card: 800x418px

#### **Web:**
- Header Logo: 200x60px (horizontal)
- Footer Logo: 150x150px (square)
- Email Signature: 300x80px

#### **Print:**
- Business Card: 300 DPI, 3.5" x 2"
- Letterhead: 300 DPI, 8.5" x 11"
- Billboard: Vector (SVG/AI)

### Formatos de Entrega:

- **Vector**: SVG, AI, EPS
- **Raster**: PNG (transparente), JPG (blanco), WebP
- **Optimizado**: Favicon.ico, Apple Touch Icon

### Variaciones Necesarias:

1. **Full Color** (gradiente teal-orange)
2. **Monochrome Teal** (#14b8a6)
3. **Monochrome Orange** (#f97316)
4. **Black** (#000000)
5. **White** (#FFFFFF)
6. **Reversed** (para fondos oscuros)

## 🎯 GUÍA DE USO POR CONTEXTO

### Para **App Icon** (iOS/Android):
- Usar solo el símbolo (sin texto)
- Fondo con gradiente o color sólido
- Bordes redondeados (sistema lo hace automáticamente)
- Alto contraste

### Para **Website Header**:
- Versión horizontal (símbolo + texto)
- Altura máxima: 60px
- Fondo transparente
- Versión dark mode (blanco)

### Para **Redes Sociales**:
- Perfil: Símbolo solo, cuadrado
- Cover/Banner: Versión horizontal con tagline
- Posts: Marca de agua sutil (esquina)

### Para **Documentos Legales**:
- Versión monocromática (negro)
- Sin gradientes
- Tipografía legible a 100%

### Para **Merchandising**:
- Versión de 1 o 2 colores (serigrafía)
- Alto contraste
- Escalable vectorialmente

## 💡 CONSEJOS PARA MEJORES RESULTADOS

### Al usar generadores de IA:

1. **Itera**: Genera 10-20 variaciones antes de elegir
2. **Refina**: Usa "remix" o "variations" para ajustar
3. **Combina**: Toma elementos de diferentes outputs
4. **Vectoriza**: Usa herramientas como Adobe Illustrator o Figma para convertir a vector
5. **Prueba**: Verifica escalabilidad (16px a 2000px)

### Validación de diseño:

- ✅ ¿Se reconoce a 16x16px (favicon)?
- ✅ ¿Funciona en blanco y negro?
- ✅ ¿Se distingue de competidores?
- ✅ ¿Representa tours virtuales + bienes raíces?
- ✅ ¿Transmite profesionalismo y tecnología?

## 🚀 SIGUIENTE PASO RECOMENDADO

### Proceso sugerido:

1. **Generar 20 conceptos** usando los 5 prompts de variantes
